# PostGIS Spatial Analysis

This project demonstrates spatial data modeling and analysis using PostgreSQL + PostGIS.

## Features
- Spatial database design
- Geometry indexing (GiST)
- Spatial queries (buffer, area calculation, intersections)
- Performance-oriented SQL

## Technologies
- PostgreSQL
- PostGIS
- SQL
- QGIS (for visualization)

## Use Cases
- Land use analysis
- Environmental monitoring
- Spatial decision support
